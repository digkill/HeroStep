# HeroStep - Пошаговая стратегия RPG для Android

Современная пошаговая стратегия RPG игра для Android, разработанная с использованием современных подходов 2025 года.

## Особенности

- **Игровое поле**: 64 шестиугольных ячейки в 3D ландшафте, соединенные как соты
- **6 рас**: Люди, Орки, Эльфы, Дворфы, Легион Хаоса, Нежить
- **3D герои и юниты**: Движение по гексагональному полю
- **Разнообразные типы местности**: Лес, Равнина, Горы, Озеро, Пустошь, Оскверненная земля
- **Таверны**: Найм юнитов за золото
- **Мобы**: Сражения с получением опыта, золота и лута
- **Система предметов и навыков**: Развитие героя
- **Однопользовательский режим**: Сражения с AI противниками

## Технологии

- **Kotlin** - основной язык разработки
- **Jetpack Compose** - современный UI фреймворк
- **Material Design 3** - дизайн система
- **Navigation Compose** - навигация между экранами
- **OpenGL ES 2.0** - 3D графика для рендеринга гексагонального поля
- **GLSurfaceView** - интеграция OpenGL с Compose

## Игровой процесс

1. **Сплеш экран** - загрузка приложения
2. **Видео заставка** - введение в игровой мир
3. **Главное меню** - выбор начала игры
4. **Выбор расы и героя** - настройка персонажа
5. **Игровое поле** - основная игровая механика

## Игровая механика

- **3D Ландшафт**: Игровое поле представлено в 3D с разными высотами для разных типов местности
- **Гексагональная сетка**: Шестиугольники соединены как соты, создавая естественное игровое поле
- **Движение**: Герои перемещаются по гексагональному полю с учетом типа местности
- **Бой**: Сражения с мобами за опыт и золото
- **Найм юнитов**: В тавернах можно нанимать юнитов за золото
- **Развитие**: Герои получают опыт, повышают уровень и улучшают характеристики
- **Предметы**: Сбор и использование предметов для улучшения героя
- **Управление камерой**: Свайпы для вращения камеры вокруг игрового поля

## Сборка проекта

1. Откройте проект в Android Studio
2. Синхронизируйте Gradle зависимости
3. Запустите на эмуляторе или устройстве Android (минимум API 26)

## Требования

- Android SDK 26+
- Kotlin 1.9.20+
- Android Studio Hedgehog или новее
